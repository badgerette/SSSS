When setting up a Roll20 game, for character sheet template select: Custom 

On your game page, go to setting and click Game Settings 
paste the SSSS HTML file into the HTML tab under your Custom character sheet template
Under the CSS tab paste the SSSS CSS
Then Save

To use the API you need a Roll20 Pro subscription.
On your game page click settings and click Mod (API) Scripts
Add a New Script
Paste the SSSS API.Json into the new script
Then Save
